## Deployment Guide

### Prerequisites
- Docker & Docker Compose v2
- Python 3.12+
- 16GB+ RAM
- 50GB disk space

### Quick Start
```bash
# Clone repository
git clone https://github.com/dwilli15/OsMEN.git
cd OsMEN

# Setup environment
cp .env.example .env
# Edit .env with your credentials

# Install dependencies
pip install -r requirements.txt

# Start services
docker compose up -d

# Verify installation
python check_operational.py
python test_agents.py
```

### Verification
1. All services running: `docker compose ps`
2. All tests passing: `python test_agents.py`
3. Integration tests: `python test_mcp_integration.py`
4. Innovation agent: `python test_innovation_agent.py`
5. Security scan: `python scripts/automation/validate_security.py`
