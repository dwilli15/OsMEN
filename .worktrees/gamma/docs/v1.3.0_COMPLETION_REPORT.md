## Architecture

```
┌─────────────────────────────────────────────────────────┐
│                  Innovation Agent System                 │
└─────────────────────────────────────────────────────────┘

Weekly Scan (GitHub Actions - Sundays 6 PM UTC)
  │
  ├─→ Feed Monitor (monitor_feeds.py)
  │     ├─ RSS/Atom feeds (7+ sources)
  │     ├─ Relevance filtering
  │     └─ Deduplication via cache
  │
  ├─→ Scoring Engine (scoring.py)
  │     ├─ Relevance: 1-10 (grad school focus)
  │     ├─ Complexity: 1-10 (integration effort)
  │     ├─ Impact: 1-10 (UX improvement)
  │     ├─ Risk: low/medium/high
  │     └─ No-Code: boolean
  │
  ├─→ Validation (validation.py)
  │     ├─ Field validation
  │     ├─ Score range checks
  │     └─ Audit logging (JSONL)
  │
  ├─→ Digest Generator (generate_digest.py)
  │     ├─ Categorize: approve/research/reject
  │     ├─ Generate markdown report
  │     └─ Update INNOVATION_BACKLOG.md
  │
  └─→ Notification System (notifications.py)
        ├─ Email weekly digest
        ├─ Approval requests
        └─ Implementation alerts

Daily Maintenance (GitHub Actions - Daily 6 PM UTC)
  │
  └─→ Daily Digest (daily_digest.py)
        ├─ Activity timeline
        ├─ Audit trail analysis
        └─ System status

Implementation Workflow
  │
  ├─→ Implementation Queue (implementation_queue.py)
  │     ├─ Priority management (critical/high/medium/low)
  │     ├─ Status tracking (queued → in-progress → completed)
  │     └─ Metrics (avg implementation time, completion rate)
  │
  └─→ Execution & Completion
        └─ Completion notifications
```
